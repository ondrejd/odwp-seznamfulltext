== Description ==

Plugin pro WordPress, který umožňuje snadnější registraci obsahu v rámci fulltext vyhledávače Seznam.cz.

Hlavní vlastnosti:

* široké možnosti nastavení pro registraci obsahu v rámci vyhledávače Seznam.cz
* použití `wp_cron` namísto okamžité registrace pro zrychlení funkce na hodně používaných webech
* stránka se zprávou o aktuálním stavu registrace obsahu


== Installation ==

Plugin nainstalujte následujícím způsobem:

1. Nahrajte adresář s pluginem do složky `/wp-content/plugins/` vaší instalace systému WordPress
2. Aktivujte plugin prostřednictvím stránky `Pluginy` v administraci systému WordPress
